# Catnip UI

An opinionated component library for ashleyndavid.com


## Lint and Format

```bash
yarn format && yarn lint
```

## Storybook

Build Storybook

```bash
yarn build-storybook
```

Run Storybook app to preview components

```bash
yarn storybook
```
